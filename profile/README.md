## 💡 关于我们

SR思锐团队（常设核心为 **思锐工作室 / SR-Studio**）成立于 2019 年。作为一支非营利性的青少年技术学习与协作社区，我们坚持**以大多数用户为中心，探索并创造有温度的产品**。我们并非谋求短期的经济回报，而是致力于让更多人真切地感受到科技带来的幸福感与美好。

### 核心使命与愿景
* **终身使命**：以大多数用户为中心，创造有温度的产品。
* **团队愿景**：为民之所好，为众之所爱，坚持探索科技创新，我们只为更好。
* **长远目标**：从当前的青少年创业实践社区，逐步成长为一家能够让更多人感受到科技带来幸福感、具有温度的科技领军企业。

## 🔮 团队文化与精神

在SR思锐，我们倡导健康、开放、包容的极客氛围：

* **团结 (Unity)**：像分布式系统一样，独立运行但紧密协作。
* **包容 (Inclusivity)**：尊重不同的技术栈与声音，不拘泥于单一技术路径。
* **互助 (Mutual Aid)**：在 Code Review 中与社区共同成长，亦师亦友。
* **共赢 (Win-Win)**：在成就他人的同时，也成就更好的自己。
* ~~**极客与二次元 (Geek & ACGN)**~~：保持对二次元文化的热爱，同时秉持极致的极客钻研精神。

## 🌐 核心生态与公共基础设施

我们正在构建以 **思锐工作室 (SR-Studio)** 为核心的青少年生态体系，并提供了以下公共技术基础设施：

### 🔑 United Pass 统一验证平台
* **定位**：由团队推出的统一身份认证系统（简称 **SRID** / **SRInternet ID**），通过一个账户即可无缝畅行我们团队现在及未来推出的大多数产品与服务。
* **隐私与安全底线**：
  * **少必要原则**：我们克制地处理数据，仅收集实现统一登录所必需的信息。
  * **数据加密**：敏感信息均经过不可逆加密（哈希）处理。
  * **访问地址**：[united.sr-studio.cn](https://united.sr-studio.cn)

我们鼓励技术的正向交流与合理衍生，并团队内部的根据[《思锐工作室 - 开源协议》（SR Open Source License）](https://sr-studio.feishu.cn/wiki/GwKxwB1Ili5bP1kNcUvcFjI5n5c#share-CruedZzNpo4KCvxOLeHck1UxnTs)对本组织名下的**部分开源资产**作出如下规范：

1. **原创指向**：思锐团队开源产品的原创作者身份始终指向 **SR思锐团队**，任何二次分发或修改版本需在代码及产品显著位置标注原创者真实有效信息。
2. **非商用默认原则**：本团队**未声明采用特定国际开源许可证（如 MIT/GPL/Apache 等）**的原创代码、美术资源及文档，**默认禁止商用**（包括但不限于接单外包、付费教学、打包出售等获利行为）。个人学习、非营利性比赛、学术交流等不属于商业用途。
3. **标准许可证优先**：若具体项目仓库中已明确声明采用国际公认开源许可证（如 **GPLv3** 等），则在您完全遵守该许可证（如开源回馈等）的前提下，**以该国际许可证的规定为最高准则**。
4. **免责与规范**：所有开源代码均按“现状（As-Is）”提供。任何个人或组织不得利用我们的代码从事违法活动、损害他人利益或进行恶意网络攻击。

*有关品牌与身份保护、资产划分、详细条款请参阅各项目仓库中的规范文档。*
